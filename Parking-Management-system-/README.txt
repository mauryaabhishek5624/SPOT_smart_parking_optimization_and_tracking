# 🚗 SPOT - Smart Parking Optimization and Tracking

A Flask + MySQL based Smart Parking Management System that automates vehicle entry, slot allocation, valet queue management, billing, and parking history.

## ✨ Features
- Vehicle Entry & Exit
- Automatic Slot Allocation
- Valet Queue (FIFO)
- Parking Billing
- Real-time Slot Status
- Parking History
- MySQL Database Integration

## 🛠️ Tech Stack
- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## 🚀 Installation

```bash
pip install -r requirements.txt
python app.py
